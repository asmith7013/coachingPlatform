✅ ANSWER
Because you can only add fractions when they have the SAME DENOMINATOR. Since 8/3 already has a denominator of 3, we convert 2 into thirds (2 = 6/3). Now we can add: 8/3 + 6/3 = 14/3. Each loot box originally cost $14/3 ≈ $4.67!

Step 1: COMPARE the x-coefficients

Daily Reward Equation

5x + 12 = 5x + ___

Both sides already have 5x — the coefficients MATCH!
What I write in the blank determines the number of solutions.

If I write a constant...

Coefficients stay matched → no solution OR infinite

If I write an expression with x (like 2x)...

Creates a MISMATCH → one solution

✅ ANSWER
Because matching coefficients means the x-terms will CANCEL when we simplify—leaving only constants to compare. Different coefficients give us a normal equation to solve!

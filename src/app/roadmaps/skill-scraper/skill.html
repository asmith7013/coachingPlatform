Description
I will recognize independent and dependent events and calculate the probabilities of compound events.
Skill Challenge Criteria
Compound events can include both independent and dependent events. Probabilities may be expressed as ratios, fractions, decimals, or percents.
Essential Question
How can I find the probability of compound events?

Compound Probability

Launch:

Ice Cream Party: You can use construction paper, props, or illustrations to show the options. Students can choose (only one selection for each of the options):

    • a cone or a cup
    • vanilla, chocolate, or strawberry ice cream 
    • sprinkles, cookie crumbs, chocolate chips, or no toppings

Start by asking what students notice and wonder about the variety of choices. Then, once each student has chosen an ice cream creation, discuss how many possible outcomes there are, and if an order were to be made randomly, how likely would it turn out to be your choice?

Teacher/Student Strategies:

Suggested Strategy for an Initial Lesson: Counting/Ratios. In this skill, students are identifying the total number of possible outcomes, and the number of favorable outcomes. The number of favorable outcomes is written as the numerator and the number of total outcomes as the denominator. 

  • Reasonableness. Prior to identifying the specific probability, students can consider if the outcome of a compound event is impossible, certain, or somewhere in between such as unlikely, even, or likely. Math is Fun shows how probabilities can be represented on a line from an ‘impossible’ event to a ‘certain’ event. This can help students to predict and/or check to make sure the probability they determined is accurate.
  • Multiplying. This skill should not focus on multiplying each event's likelihood, as this strategy is not a focus until high school, but if a student is already familiar, they can use it.

Models and Manipulatives:

Highlight Model: Tree Diagram. Students should be familiar with tree diagrams from skill 166. The probability of an outcome from a compound event can be represented as a fraction, decimal, ratio, or percent.

 

  • Area Model or Array. Students can determine probability using an area model similarly to how they use the tree diagram. The total number of square units is the total number of possible outcomes, becoming the denominator. Then the number of favorable outcomes becomes the numerator. 

  • Organized List. Students may also list all the possible outcomes, though this model can be more error-prone for long lists.

Questions to Help Students Get Unstuck:

  • Read the context again. What’s one possible outcome? How many others do you think there are? 
  • We can use a tree diagram or an array to start. Which one do you want to draw?
  • How many of the total possible outcomes are favorable?

    • Discussion Questions:

    • If your teacher has the name of each student in a hat and draws one at random, what is the probability of your name being chosen?
    • Why is the probability of pulling a given colored marble from a bag different if you don’t return each marble pulled than if you do?
    • What makes one event dependent on another event? What makes it independent?

    Common Misconceptions:

    • Students may determine the probability as if an event is independent, when it is actually dependent. 
    • Students may overlook repeated outcomes (for example, if a spinner has multiple sections of the same color) and end up with an incorrect probability.
    • When writing probability as a fraction, students may identify the numerator correctly as the number of favorable outcomes, but misidentify the denominator as the number of unfavorable outcomes rather than as the total number of possible outcomes.

    Additional Resources:

    • Math is Fun explains events and outcomes as well as when events are dependent and independent. (Mutually exclusive events are not part of this skill.) Math is Fun also includes examples of how you can use tree diagrams to find the probability of an event, and an activity including two dice to model compound events.
    • You can use an interactive spinner from Math is Fun, or set of dice from Random.org, coins from Shodor, or marbles from Shodor to model compound events.
    • This video about the lottery from PBS Learning Media dives into a single real-world scenario, which might be a good introduction for students. There is another one about a deck of cards.

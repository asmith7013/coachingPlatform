# Double Number Line Pattern

## Overview

Two parallel number lines showing proportional relationships with input boxes for student interaction.

## Visual Structure

```
      Days:    0    3    6    9   [__]  [__]
            ---|----|----|----|----|----|
      Pies:    0    8   16   24   [__]  [__]
            ---|----|----|----|----|----|
```

## When to Use

- Equivalent ratios
- Proportional relationships
- Rate visualization
- Completing ratio tables visually
